# 🚀 BugBook - Social Media Platform

A modern full-stack social media web application built using **Next.js 15**, **React 19**, **TypeScript**, **Prisma**, **Neon PostgreSQL**, **Lucia Authentication**, **Stream Chat**, and **UploadThing**.

This project was developed as a **Full Stack Development Internship Task** for **CodeAlpha**.

# Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_Social-Media.git
```

---

## Navigate

```bash
cd CodeAlpha_Social-Media
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env` file and configure:

```env
POSTGRES_PRISMA_URL=
POSTGRES_URL_NON_POOLING=

NEXT_PUBLIC_BASE_URL=

AUTH_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=

NEXT_PUBLIC_STREAM_KEY=
STREAM_SECRET=
```

---

## Generate Prisma Client

```bash
npx prisma generate
```

---

## Run Database Migration

```bash
npx prisma db push
```

---

## Start Development Server

```bash
npm run dev
```
